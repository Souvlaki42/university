# Περιγραφή κίνησης χαρακτήρων και κανόνες

## Εκτέλεση

- Εκκίνηση: `./final map1.txt` ή με δυαδικό χάρτη `map2.dat`.
- Τερματισμός: πατήστε ESC.

## Κίνηση χαρακτήρων

Οι χαρακτήρες κινούνται αυτόματα, ακολουθώντας τα εξής βήματα:

Αν υπάρχει στόχος (κλειδί, κλουβί, σκάλα) σε διπλανό κελί, κινούνται απευθείας εκεί.

Αν δεν υπάρχει στόχος σε διπλανό κελί, συνεχίζουν ευθεία αν γίνεται, αλλιώς επιλέγουν ένα από τα 4 γειτονικά κελιά με τα λιγότερα περάσματα.
Κάθε κίνηση ενημερώνει τον μετρητή επισκέψεων του κελιού.

Οι χαρακτήρες "βλέπουν" μόνο τα 4 γειτονικά κελιά (όχι διαγώνια).

## Λήξη

- Χάνει: αν και οι δύο παγιδευτούν, ή κάποιος παγιδευμένος κρατά κλειδί, ή εξαντληθούν τα μέγιστα βήματα.
- Νικά: αν και οι δύο φτάσουν στη σκάλα μετά τη συνάντηση.
